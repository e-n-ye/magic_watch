# v0.8c-2 Sleep 说明正文闭环

日期：2026-05-24

## 本轮目标

把 `SleepInfoPage` 从“说明入口壳层”推进成“真实可滚动正文页”。

本轮只解决一件事：

- `Sleep -> 说明` 进入后，页面有真实中文正文内容，并支持滚动阅读。

## 为什么这轮单独拆出来

如果把 `Sleep` 说明正文和睡眠数据卡、近 7 天图表、昨晚详情、真实睡眠模型放在同一轮，会重新把范围拉大。

这轮只做正文页有两个好处：

1. 保持 `Sleep` 这条线继续按小闭环推进。
2. 让 `说明` 页不再是占位壳层，减少主路径里的半成品感。

## 本轮范围

### 1. 页面标题

- 顶部标题从通用 `说明` 收成更明确的 `睡眠说明`

### 2. 正文内容

- 根据参考图补入真实正文
- 保持中文长段落可滚动阅读
- 不增加图标卡片、图表或其他附加模块

### 3. 交互保持不变

- 顶部返回按钮仍回到 `Sleep` 首页
- 表冠滚动仍然可用
- 页面仍然通过现有 `scroll_root_` 承载正文

## 本轮明确不做

- 不新增睡眠数据模型
- 不补“昨晚睡眠记录”真实内容
- 不补“近 7 天睡眠”真实图表
- 不改 `Sleep 设置`

## 验收标准

1. 从 `AppSleep` 进入 `说明`
2. 页面标题显示 `睡眠说明`
3. 页面正文为真实中文内容，不再是占位文案
4. 触摸滚动和表冠滚动都能浏览完整正文
5. 返回路径正常

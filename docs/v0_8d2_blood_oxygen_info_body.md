# v0.8d-2 Blood Oxygen 说明正文闭环

日期：2026-05-25

## 本轮目标

把 `BloodOxygenInfoPage` 从“说明入口壳层”推进成“真实可滚动正文页”。

本轮只解决一件事：

- `AppBloodOxygen -> 说明` 进入后，页面有真实中文正文内容，并支持滚动阅读。

## 为什么这轮单独拆出来

`v0.8d-1` 已经完成 `AppBloodOxygen` 首页、设置页和说明壳层。继续推进时，如果把说明正文、真实血氧测量、低血氧提醒通知和历史趋势放进同一轮，会重新扩大范围。

这轮只补说明正文，能让主路径不再有明显占位感，同时保持 Blood Oxygen 线仍然按小闭环推进。

## 本轮范围

### 1. 正文内容

- 根据参考图补入血氧饱和度说明
- 包含血氧饱和度定义、参考值、影响因素、测量佩戴要求和非医疗器械提示
- 保持中文长段落可滚动阅读

### 2. 交互保持不变

- 顶部标题继续使用 `血氧说明`
- 顶部返回按钮仍回到 `AppBloodOxygen` 首页
- 表冠滚动仍然可用
- 页面仍然通过现有 `scroll_root_` 承载正文

## 本轮明确不做

- 不接真实血氧测量数据源
- 不接真实“开始测量”流程
- 不做低血氧提醒通知链路
- 不新增血氧历史趋势或数据模型
- 不新增图示绘制

## 验收标准

1. 从 `AppBloodOxygen` 进入 `说明`
2. 页面标题显示 `血氧说明`
3. 页面正文为真实中文内容，不再是占位文案
4. 正文包含血氧参考值、影响因素、测量佩戴要求和非医疗器械提示
5. 触摸滚动和表冠滚动都能浏览完整正文
6. 返回路径正常

# v0.8e-3 AppHeartRate 心率说明正文

日期：2026-05-25

## 本轮目标

在 `v0.8e-1` 主页面与 `v0.8e-2` 心率设置通过手工验收后，补齐 `心率说明` 正文。

本轮只做说明页可达与正文内容，不扩真实测量、提醒通知或未佩戴算法。

## 本轮范围

### 1. 说明入口

- `AppHeartRate` 主页面中的 `心率说明` 从占位入口变为真实可点击入口
- 点击后进入 `心率说明`
- 支持顶部返回、触摸滚动和表冠滚动

### 2. 说明正文

正文覆盖参考图中的核心内容：

- 心率定义：心率是心脏每分钟跳动的次数（bpm）
- 安静状态下成年人静息心率通常为 `55-80次/分`
- 个体差异可能来自年龄、性别、健身水平或其他生理因素
- 设备测量范围为 `30-240次/分`
- 测量说明：
  - 将设备佩戴至腕骨一指距离，约 `1.5cm-2cm`
  - 偏紧佩戴，测量期间保持静止
- 佩戴状态、皮肤特征、环境等因素可能影响测量准确性，甚至无法输出结果
- 本品非医疗器械，结果仅供参考，不作为诊断治疗依据

## 本轮明确不做

- 不接真实心率传感器
- 不扩心率数据模型
- 不实现高低心率提醒通知链路
- 不实现未佩戴检测算法
- 不改心率设置页逻辑

## 验收标准

1. 从 `AppHeartRate` 点击 `心率说明`
2. 页面标题显示 `心率说明`
3. 正文可见 `心率`、`55-80次/分`、`30-240次/分`、`测量说明`、`1.5cm-2cm`、`本品非医疗器械`
4. 页面支持触摸滚动和表冠滚动
5. 顶部返回按钮回到 `AppHeartRate`
